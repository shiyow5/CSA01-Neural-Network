## a) 解いた問題

パート2の目的は、パート1の単一ニューロン学習を複数の出力ニューロンを持つ**単層ニューラルネットワーク**に拡張することである。課題は、3つの入力パターンを3つの異なるクラスに分類する**3クラス分類問題**である。

ネットワーク構造:
- **入力ニューロン (N):** 3（データ入力2 + バイアス入力 -1）
- **出力ニューロン (R):** 3（各クラスに1つ）

訓練データは3サンプルで構成される:

| サンプル | 入力 x1 | 入力 x2 | バイアス | 期待出力 (o1, o2, o3) |
|:------:|:------:|:------:|:------:|:--------------------:|
| 1      | 10     | 2      | -1     | (+1, -1, -1)         |
| 2      | 2      | -5     | -1     | (-1, +1, -1)         |
| 3      | -5     | 5      | -1     | (-1, -1, +1)         |

各出力ニューロンは1つのクラスを表す。正しいクラスに対して+1、それ以外のクラスに対して-1を出力する（{-1, +1}によるワンホットエンコーディング）。

## b) 使用した手法

### パーセプトロン学習則（単層ネットワーク）

R=3の各出力ニューロンはそれぞれN=3の重みを持つ。活性化関数としてステップ関数を使用する:

```
f(x) = +1  (x >= 0 の場合)
f(x) = -1  (x < 0 の場合)
```

各出力ニューロン i に対する重みの更新則:

```
w_ij(t+1) = w_ij(t) + eta * (d_i - o_i) * x_j
```

ここで `w_ij` は入力 j から出力ニューロン i への重みである。合計誤差は全出力ニューロンおよび全サンプルにわたる二乗誤差の和として計算される。

### デルタ学習則（単層ネットワーク）

デルタ学習則はシグモイド活性化関数を用いて多出力の場合に拡張される:

```
f(x) = 2 / (1 + exp(-lambda * x)) - 1,  ここで lambda = 1.0
```

各出力ニューロン i に対する重みの更新:

```
delta_i = (d_i - o_i) * (1 - o_i^2) / 2
w_ij(t+1) = w_ij(t) + eta * delta_i * x_j
```

各出力ニューロンは専用の重みベクトルを通じて独立に自身の決定境界を学習する。

### 共通設定

- 学習率 (eta): 0.5
- 収束閾値: 0.1
- 重みは [-0.5, 0.5) の範囲でランダムに初期化

## c) シミュレーション結果に関する考察

### パーセプトロン学習の結果

パーセプトロン学習則は **4 エポック** で収束した。最終的な重み行列:

```
ニューロン1: [4.121,  4.974,  4.615]
ニューロン2: [-2.803, -12.022, 1.111]
ニューロン3: [-9.803, -1.609,  0.993]
```

全入力パターンに対する最終出力:

| 入力             | 出力              | 期待値             |
|:---------------:|:-----------------:|:-----------------:|
| (10, 2, -1)     | (+1, -1, -1)      | (+1, -1, -1)      |
| (2, -5, -1)     | (-1, +1, -1)      | (-1, +1, -1)      |
| (-5, 5, -1)     | (-1, -1, +1)      | (-1, -1, +1)      |

全てのパターンが正確な +1/-1 の出力で完全に分類された。

### デルタ学習の結果

デルタ学習則は **3 エポック** で収束した。最終的な重み行列:

```
ニューロン1: [2.224,  1.556, -0.004]
ニューロン2: [-1.323, -2.837, 0.455]
ニューロン3: [-0.485,  0.392, 0.249]
```

全入力パターンに対する最終出力:

| 入力             | 出力                         | 期待値             |
|:---------------:|:---------------------------:|:-----------------:|
| (10, 2, -1)     | (+1.000, -1.000, -0.974)    | (+1, -1, -1)      |
| (2, -5, -1)     | (-0.931, +1.000, -0.920)    | (-1, +1, -1)      |
| (-5, 5, -1)     | (-0.931, -0.999, +0.968)    | (-1, -1, +1)      |

全ての出力が期待値に近い値となった。正しいクラスのニューロンが常に最も高い出力値を持つ。

### 比較と考察

1. **収束速度:** 本問題では、デルタ学習則（3エポック）がパーセプトロン（4エポック）よりわずかに速く収束した。これはパート1でパーセプトロンの方がはるかに速かったのとは対照的である。理由として、収束閾値がより緩い（0.01ではなく0.1）こと、またシグモイド関数が全ニューロンの誤差を同時に漸減させることができる一方、パーセプトロンの離散的な出力は異なるパターン間で誤差の振動を引き起こす可能性があることが挙げられる。

2. **分類精度:** 両手法とも3つの入力パターン全てを正しく分類した。パーセプトロンの出力は正確な+1/-1であるのに対し、デルタ学習の出力は近似値だが目標に近い（例: -1の代わりに-0.974）。

3. **重みの構造:** パーセプトロンはデルタ学習則（最大約3）と比べてより大きな重みの絶対値（最大12）を生成する。パーセプトロンは完全な誤差信号 `(d-o)`（0または+/-2）で重みを調整するため、より大きな重みが必要となる。デルタ則はスケーリングされた勾配を使用し、より穏やかな重み更新を行う。

4. **入力スケールの影響:** パート2の入力データはパート1（2値 0/1）と比べてはるかに大きな値（10, -5 など）を持つ。これにより1回の反復あたりの重み更新が大きくなり、両手法ともより速い収束につながる。クラス重心間の大きな距離も、分類問題を比較的容易にしている。

5. **ネットワークの拡張性:** パート2は、両学習則が単一ニューロンから多ニューロンネットワークに自然に拡張できることを示している。各出力ニューロンは独立に自身の重みを学習し、実質的にR個の独立した二値分類問題を並列に解いている。

6. **単層ネットワークの限界:** 両手法ともこの線形分離可能な3クラス問題では良好に動作するが、単層ネットワークは非線形分離問題（例: XOR）を解くことができない。そのような問題には、隠れ層と誤差逆伝播法を持つ多層ネットワークが必要となる。

7. **実用上の考慮事項:** デルタ学習則は近似的な出力を生成するものの、微分可能なシグモイド活性化によりより滑らかな最適化ランドスケープを提供する。この特性は、勾配情報が複数の層を通して伝播する必要がある多層アーキテクチャへの拡張において極めて重要となる。

## d) 新しい問題: より近いクラス中心

### 問題の説明

クラス中心間の距離が両学習則の収束挙動にどう影響するかを調査した。漸進的に近づけた4つのデータセットを検証:

| データセット    | クラス1中心 | クラス2中心 | クラス3中心 |
|:------------:|:----------:|:----------:|:----------:|
| 原問題（遠い）  | (10, 2)    | (2, -5)    | (-5, 5)    |
| 中程度         | (3, 1)     | (1, -2)    | (-2, 2)    |
| 近い           | (1.5, 0.5) | (0.5, -1.0)| (-1.0, 1.0)|
| 非常に近い     | (1.0, 0.3) | (0.3, -0.5)| (-0.5, 0.5)|

全データセットで同じ期待出力（ワンホットエンコーディング）と同じ学習パラメータ（eta = 0.5, 閾値 = 0.1）を使用。各実験は異なるランダムシードで10回の試行を実施。

### 結果

| データセット    | パーセプトロン平均エポック | デルタ平均エポック |
|:------------:|:---------------------:|:----------------:|
| 原問題（遠い）  | 3.2                   | 11.9             |
| 中程度         | 3.0                   | 9.8              |
| 近い           | 2.8                   | 27.6             |
| 非常に近い     | 3.4                   | 79.7             |

全試行が収束した（収束率 100%）。

### 分析

1. **パーセプトロンの頑健性:** パーセプトロン学習則はクラス中心間距離に対して驚くほど鈍感である。クラスがどれだけ近くても 3〜4 エポックで収束する。ステップ関数は点が決定境界のどちら側にあるかのみを判定し、境界からの距離は関係ないためである。

2. **デルタ学習の感度:** デルタ則の収束時間はクラス中心が近づくにつれて劇的に増加する:
   - 原問題（遠い）: 11.9 エポック
   - 非常に近い: 79.7 エポック（6.7 倍遅い）
   
   クラスが近い場合、異なるパターンに対するシグモイド出力がより類似し、誤差信号と重み更新が小さくなる。類似した入力を識別できる重みを発達させるために、より多くの反復が必要となる。

3. **実用上の含意:** 実世界の分類タスクでは、クラスの分離が不十分であることは一般的である。デルタ学習則のクラス分離距離に対する感度は、効率的な訓練のために適切なデータ前処理（特徴量スケーリング、次元削減等）が重要であることを示唆する。

## e) 手法の改良: 重み初期化の影響

### 説明

重み初期化の範囲が収束挙動に与える影響を調査した。原問題（遠いクラス中心）を使用し、6つの異なる初期化範囲を検証: **[-0.01, 0.01], [-0.1, 0.1], [-0.5, 0.5], [-1.0, 1.0], [-2.0, 2.0], [-5.0, 5.0]**。各範囲について、異なるランダムシードで20回の試行を実施。

### 結果

| 初期化範囲 | デルタ平均エポック | デルタ標準偏差 | デルタ収束率 | パーセプトロン平均エポック | パーセプトロン収束率 |
|:---------:|:----------------:|:------------:|:-----------:|:---------------------:|:------------------:|
| 0.01      | 4.2              | 0.8          | 100%        | 3.2                   | 100%               |
| 0.10      | 4.0              | 1.4          | 100%        | 3.2                   | 100%               |
| 0.50      | 9.5              | 9.0          | 100%        | 3.2                   | 100%               |
| 1.00      | 87.5             | 172.4        | 95%         | 3.3                   | 100%               |
| 2.00      | 4521.1           | 9465.3       | 80%         | 2.6                   | 100%               |
| 5.00      | 7579.2           | 7051.6       | 20%         | 2.8                   | 100%               |

### 分析

1. **パーセプトロンの免疫性:** パーセプトロンは初期化範囲の影響を全く受けない。重みが 0 付近から始まっても +/-5 から始まっても、2〜3 エポックで収束する。ステップ関数の出力は重み付き和の符号のみで決まり、その大きさには依存しないためである。

2. **デルタ学習 — 決定的な感度:** デルタ則は初期化範囲に極めて敏感である:
   - **小範囲 (0.01-0.10):** 高速かつ安定した収束（約4エポック、低分散）
   - **中範囲 (0.50):** 収束するが分散が大きい（標準偏差 = 9.0）
   - **大範囲 (1.00+):** 収束が不安定に。範囲 = 5.0 では 50,000 エポック以内に収束したのは 20% のみ。

3. **飽和問題:** 大きな初期重みはシグモイド関数を飽和させ（出力が +1 または -1 付近）、導関数 `(1-o^2)/2` がゼロに近づく。これにより効果的な学習を妨げる「勾配消失」が発生する。これはニューラルネットワーク訓練における周知の問題であり、現代の深層学習における Xavier/He 初期化などの手法の動機となっている。

4. **分散の爆発:** 初期化範囲の増大に伴い、収束時間の標準偏差が爆発的に増大する（範囲 0.01 での 0.8 から範囲 2.0 での 9,465）。これは大きな初期重みによる訓練挙動の高い予測不可能性を示している。

## f) 拡張実験に関する考察

### 知見のまとめ

d) と e) の拡張実験は、単層ニューラルネットワークの訓練に関する実践的な知見を提供する:

1. **クラス分離と収束:** クラス中心が近いほど、デルタ学習則の収束が困難になる。パーセプトロンは頑健だが、それは二値判定を行うためである。実際には、デルタ則の感度は、訓練前にクラス分離を最大化するための特徴量工学やデータ正規化の使用を動機づける。

2. **重み初期化は決定的:** 勾配ベースの学習（デルタ則）では、適切な重み初期化が不可欠である。ゼロ付近の小さな初期重みは、ニューロンがシグモイドの線形領域（勾配が最大の領域）で動作することを保証する。大きな初期重みはニューロンを飽和に追いやり、勾配消失と訓練の失敗を引き起こす。この知見は、深層学習で標準的な Xavier 初期化や He 初期化などの現代的な初期化手法に直接つながるものである。

3. **パーセプトロン vs デルタのトレードオフ:** パーセプトロン学習則は、単純な線形分離可能問題に対してハイパーパラメータやデータ特性への頑健性が高い。しかし、非微分なステップ関数のため多層ネットワークでは使用できない。デルタ則はハイパーパラメータに対してより敏感だが、より深いアーキテクチャの訓練に必要な勾配情報を提供する。
